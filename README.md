# Neural Network in C++ (Currently in Progress)

A simple neural network implemented in C++ with a modern CMake build system.

## Requirements
* CMake ≥3.20
* C++20-compatible compiler (works with Clang and GCC)

## Build Instructions

Clone the repository and build using CMake:

```bash
git clone git@github.com:ezrazka/neural-network-in-cpp.git
cd neural-network-in-cpp
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
./build/neuralnetwork
```
